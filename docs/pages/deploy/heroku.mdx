# Heroku Deploy

Heroku is no longer a first-class deployment target for this repository.

## Why

The project now assumes:

- a root-first runtime layout
- `uv` for dependency management
- a Docker-first deployment story

The old Heroku-specific runtime files have been removed from the repository.

## Recommendation

If you want the simplest hosted path, use Railway or Render from the current docs instead.
